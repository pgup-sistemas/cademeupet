## 📊 FLUXOS AGORA FUNCIONAIS

### ✅ Fluxo PIX
```
Usuário → Preenche formulário e seleciona PIX
        → DoacaoController registra doação
        → PagamentoController cria cobrança PIX
        → SDK EFI retorna TXID + QR Code
        → Usuário é redirecionado
        → Usuário escaneia QR Code
        → Depois do pagamento, webhook processa
        → Status automaticamente atualizado para "aprovada"
```

### ✅ Fluxo Cartão (À Vista)
```
Usuário → Preenche formulário e seleciona Cartão
        → DoacaoController registra doação
        → PagamentoController cria link de pagamento
        → SDK EFI retorna payment_url
        → Usuário é redirecionado para link
        → Preenche dados do cartão
        → Webhook processa o resultado
        → Status automaticamente atualizado
```

### ✅ Fluxo Cartão (Mensal/Assinatura)
```
Usuário (logado) → Seleciona Cartão Mensal
                → PagamentoController cria plano + assinatura
                → SDK EFI retorna link de assinatura
                → Usuário autoriza débito mensal
                → Primeira cobrança é processada
                → Webhook marca como "aprovada"
                → Próximo mês: cobrança automática
```

## 🚀 COMO USAR AGORA

### PASSO 1: Verificação Rápida
```bash
# Via terminal
php quick_check.php

# Ou via browser
https://petfinder.pageup.net.br/quick_check.php
```

### PASSO 2: Configurar Credenciais
Editar `.env` e preencher:
```env
EFI_CLIENT_ID=seu_client_id_aqui
EFI_CLIENT_SECRET=seu_client_secret_aqui
EFI_PIX_KEY=sua_chave_pix@email.com
EFI_SANDBOX=true  # Para testes
```

### PASSO 3: Instalar SDK
```bash
composer require efipay/sdk-php-apis-efi
```

### PASSO 4: Obter Certificado
1. Fazer login na conta EFI
2. Baixar certificado SSL/TLS
3. Colocar em `certs/production.pem`

### PASSO 5: Testar
```
https://petfinder.pageup.net.br/test_fluxo_doacao.php
```

### PASSO 6: Validar
1. Configurar webhooks na conta EFI
2. Testar fluxo PIX e Cartão
3. Remover `test_fluxo_doacao.php`
4. Setar `EFI_SANDBOX=false`
