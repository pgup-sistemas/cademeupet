📋 FLUXOS FUNCIONAIS
═════════════════════════════════════════════════════════════════════════════════

✅ PIX
  Usuário escaneia QR Code → Paga → Webhook processa → Status = "aprovada"

✅ CARTÃO (À VISTA)
  Usuário preenche dados cartão → Paga → Webhook processa → Status = "aprovada"

✅ CARTÃO (MENSAL/ASSINATURA)
  Usuário autoriza débito mensal → 1ª cobrança → Aprovada
  → Próximo mês: Cobrança automática → Aprovada

⚙️ CONFIGURAÇÕES NECESSÁRIAS
═════════════════════════════════════════════════════════════════════════════════

.env
────
EFI_CLIENT_ID=seu_client_id_aqui
EFI_CLIENT_SECRET=seu_client_secret_aqui
EFI_PIX_KEY=sua_chave_pix@email.com
EFI_SANDBOX=true  (mudar para false em produção)
EFI_CERTIFICATE_PATH=/caminho/para/certificado.pem
EFI_WEBHOOK_TOKEN=seu_token_secreto_aqui

SDK
────
$ composer require efipay/sdk-php-apis-efi

Certificado
────────────
1. Acessar https://dashboard.efipay.com.br
2. Baixar production.pem
3. Colocar em certs/production.pem

Webhooks (na conta EFI)
───────────────────────
PIX: https://seusite/api/efi-webhook.php?token=SEU_TOKEN
Billing: https://seusite/api/efi-billing-notification.php?token=SEU_TOKEN

❌ ERROS COMUNS
═════════════════════════════════════════════════════════════════════════════════

"SDK Efi\EfiPay não encontrada"
→ $ composer require efipay/sdk-php-apis-efi

"Credenciais EFI não configuradas"
→ Preencher EFI_CLIENT_ID e EFI_CLIENT_SECRET em .env

"Certificado EFI não encontrado"
→ Colocar production.pem em certs/

"Webhook não recebe notificações"
→ Configurar em https://dashboard.efipay.com.br

"Doação fica pendente após pagamento"
→ Ver logs em includes/petfinder_error_log
→ Verificar se webhooks estão sendo recebidos
